## Setup

use `Symlink` link `~/.config/nvim` to `~/doctfiles/nvim`

```bash
mkdir ~/doctfiles/nvim
mv ~/.config/nvim/* ~/dotfiles/nvim/
rm -r ~/.config/nvim
ln -s ~/dotfiles/nvim ~/.config/nvim
```
