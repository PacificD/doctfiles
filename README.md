## Setup

use `Symlink` link `~/.config/nvim` to `doctfiles/nvim`

```bash
cp -r ~/.config/nvim ~/.config/nvim.bak
rm -rf ~/.config/nvim
cd dotfiles
ln -s $PWD/nvim ~/.config/nvim
```
